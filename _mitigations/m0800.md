## Description

The device or system should restrict read, manipulate, or execute privileges to only authenticated users who require access based on approved security policies. Role-based Access Control (RBAC) schemes can help reduce the overhead of assigning permissions to the large number of devices within an ICS. For example, IEC 62351 provides examples of roles used to support common system operations within the electric power sector [International Electrotechnical Commission July 2020](https://webstore.iec.ch/publication/6912) , while IEEE 1686 defines standard permissions for users of IEDs. [Institute of Electrical and Electronics Engineers January 2014](https://standards.ieee.org/standard/1686-2013.html)
