## Description

Adversaries may abuse Microsoft Outlook forms to obtain persistence on a compromised system. Outlook forms are used as templates for presentation and functionality in Outlook messages. Custom Outlook forms can be created that will execute code when a specifically crafted email is sent by an adversary utilizing the same custom Outlook form. [SensePost Outlook Forms](https://sensepost.com/blog/2017/outlook-forms-and-shells/) 

Once malicious forms have been added to the user’s mailbox, they will be loaded when Outlook is started. Malicious forms will execute when an adversary sends a specifically crafted email to the user. [SensePost Outlook Forms](https://sensepost.com/blog/2017/outlook-forms-and-shells/)

- [Behavior Prevention on Endpoint](/mitigations/M1040/) (`M1040`)
- [Update Software](/mitigations/M1051/) (`M1051`)
