## Description

Adversaries may use trusted scripts, often signed with certificates, to proxy the execution of malicious files. Several Microsoft signed scripts that have been downloaded from Microsoft or are default on Windows installations can be used to proxy execution of other files. [LOLBAS Project](https://github.com/LOLBAS-Project/LOLBAS#criteria) This behavior may be abused by adversaries to execute malicious files that could bypass application control and signature validation on systems.
