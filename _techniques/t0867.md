## Description

Adversaries may transfer tools or other files from one system to another to stage adversary tools or other files over the course of an operation. [Enterprise ATT&CK](https://attack.mitre.org/techniques/T1489/) Copying of files may also be performed laterally between internal victim systems to support Lateral Movement with remote Execution using inherent file sharing protocols such as file sharing over SMB to connected network shares. [Enterprise ATT&CK](https://attack.mitre.org/techniques/T1489/) 

In control systems environments, malware may use SMB and other file sharing protocols to move laterally through industrial networks.

### Mitigations

- [Network Intrusion Prevention](/mitigations/M0931/) (`M0931`)
