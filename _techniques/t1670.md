## Description

Adversaries may carry out malicious operations using virtualization solutions to escape from Android sandboxes and to avoid detection. Android uses sandboxes to separate resources and code execution between applications and the operating system.[Android Application Sandbox](https://source.android.com/docs/security/app-sandbox) There are a few virtualization solutions available on Android, such as the Android Virtualization Framework (AVF).[Android AVF Overview](https://source.android.com/docs/core/virtualization)  

 

Through virtualization solutions, adversaries may execute malicious operations without user knowledge. For example, adversaries may mimic a legitimate banking application’s functionalities in a virtual environment, thanks to the virtualization solution, while malicious code captures credentials.
