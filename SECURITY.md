# Security Policy

## Supported Versions

| Version | Supported          |
| ------- | -----------------|
| latest  | :white_check_mark:|

## Reporting a Vulnerability

This project contains open source threat intelligence data for educational and research purposes. 

If you discover a security vulnerability in this codebase (not in the threat data itself), please report it responsibly:

1. **Do NOT** create a public GitHub issue
2. Email security concerns to the maintainers privately
3. Include detailed steps to reproduce
4. Allow time for acknowledgment before disclosure

## Data Verification

The data in this repository is compiled from multiple public sources. While we strive for accuracy:
- Information may be outdated or incomplete
- Always verify critical intelligence independently
- Do not use for operational security decisions without validation
- External links and IOCs may change without notice
