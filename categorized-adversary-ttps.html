Pivot Count values come from the merged Categorized Adversary TTPs dataset (MITRE ATT&CK × ETDA/ThaiCERT), but only for MITRE groups (G####) that match a threat actor in this project (mitre_id / external_id). Each count is how many of those matched groups contribute that technique under the selected pivot tag (industry, motivation, or victim country). They can still differ from the ATT&CK matrix (page citations) and from technique pages (YAML citations plus group-linked techniques when G#### matches). See source attribution for licenses.
