## Introduction
Mustang Panda (also known as TA416, RedDelta, BRONZE PRESIDENT, or STATELY TAURUS) is a China-based cyber espionage threat actor that has been conducting operations since at least 2012. The group targets government, diplomatic, and non-governmental organizations, including think tanks, religious institutions, and research entities, across the United States, Europe, and Asia.

Notable activity includes operations targeting Russia, Mongolia, Myanmar, Pakistan, and Vietnam. The group is known for using tailored phishing lures and decoy documents to deliver malicious payloads, primarily using the PlugX remote access trojan.

## Activities and Tactics
Mustang Panda employs sophisticated cyber espionage tactics:

### Primary Targets

- Diplomatic organizations
- Think tanks and research institutions
- Non-governmental organizations (NGOs)
- Religious institutions

## Tactics, Techniques, and Procedures (TTPs)
Mustang Panda is known for the following TTPs:

### Initial Access
- **Spear Phishing**: Malicious attachments and links
- **Phishing**: Using tailored lures and decoy documents

### Execution
- **PowerShell**: Using PowerShell for payload execution
- **Masquerading**: Abusing legitimate-looking executables
- **Msiexec**: Living-off-the-land execution via MSI packages

### Persistence
- **Registry Run Keys**: Adding entries to HKLM Run keys
- **DLL Side-loading**: Exploiting legitimate binaries

### Defense Evasion
- **Process Injection**: Injecting into trusted processes
- **Token Manipulation**: Using token manipulation
- **String Obfuscation**: Heavy string obfuscation
- **API Hashing**: Using DJB2 and ROL-13 hashing

### Discovery
- **System Information Discovery**: Gathering system info
- **Network Configuration Discovery**: Using ipconfig, arp, route

### Lateral Movement
- **Internal Network Enumeration**: Using Adfind-like tools
- **WMI Queries**: Active Directory reconnaissance
