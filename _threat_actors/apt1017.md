## Introduction
Volt Typhoon (also known as BRONZE SILHOUETTE, Vanguard Panda, or DEV-0391) is a People's Republic of China (PRC) state-sponsored actor that has been active since at least 2021. The group primarily targets critical infrastructure organizations in the US and its territories, including Guam.

Volt Typhoon's targeting and pattern of behavior have been assessed as pre-positioning to enable lateral movement to operational technology (OT) assets for potential destructive or disruptive attacks in the event of increased geopolitical tensions. The group emphasizes stealth using web shells, living-off-the-land (LOTL) binaries, hands-on-keyboard activities, and stolen credentials.

## Activities and Tactics
Volt Typhoon employs unique stealth-focused tactics:

### Primary Targets
- Communications sector
- Transportation systems
- Water and wastewater systems
- Energy sector
- Government organizations
- Critical manufacturing

### Key Motivations
- Pre-positioning for potential destructive attacks
- Long-term persistent access
- Intelligence gathering on OT systems

### Geographic Focus
- United States (primary)
- Guam
- US territories

## Notable Campaigns
1. **Critical Infrastructure Pre-positioning (2021-present)**: Gaining access to OT systems
2. **Guam Infrastructure Targeting**: Specific targeting of Guam infrastructure
3. **Living off the Land Operations**: Using native Windows tools

## Tactics, Techniques, and Procedures (TTPs)
Volt Typhoon is known for living-off-the-land techniques:

### Initial Access
- **Exploit Public-Facing Applications**: Including zero-day vulnerabilities
- **Valid Accounts**: Using stolen credentials

### Execution
- **Living off the Land**: Using native Windows tools exclusively
- **PowerShell**: Using built-in PowerShell
- **Command Prompt**: Using cmd.exe

### Persistence
- **Valid Accounts**: Maintaining access via stolen credentials
- **Web Shells**: Using web shells for persistence

### Defense Evasion
- **Living off the Land**: Avoiding custom malware
- **Log Deletion**: Rotating or deleting logs
- **Proxy via SOHO Routers**: Using compromised routers

### Discovery
- **Network Service Discovery**
- **System Information Discovery**

### Lateral Movement
- **Remote Services**: Using native tools
- **Valid Accounts**: Credential-based movement

### Collection
- **Data from Information Repositories**

## Notable Indicators of Compromise (IOCs)
Based on CISA advisory AA24-038A:

### Key Characteristics
- No custom malware deployment
- Use of native OS tools
- Compromised SOHO routers as proxies
- Web shell deployment

### MITRE ATT&CK Techniques
Volt Typhoon emphasizes techniques including:
- T1190 - Exploit Public-Facing Application
- T1078 - Valid Accounts
- T1059 - Command and Scripting Interpreter
- T1083 - File and Directory Discovery
- T1021 - Remote Services

## Malware and Tools
Volt Typhoon typically does NOT use custom malware, relying instead on:
- Native Windows tools (cmd.exe, netsh, PowerShell)
- Compromised SOHO routers
- Web shells
- Stolen credentials
